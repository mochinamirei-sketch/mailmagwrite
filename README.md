# ハチカッテ ニュースレター作成ツール（Streamlit）

配信日・商品URL・アピールしたい方向性・レコメンドバナーを入力するだけで、
- 配信テンプレート差し込み用の **JSON**
- 配信用の **テキストメール本文**

を自動生成するツールです。商品のタイトル・本文は、商品ページの情報とAnthropic API（Claude）を使って自動で書き起こします。

## 使い方の流れ

1. 配信日を入力（キャンペーンIDは自動で作られます）
2. 紹介したい商品のURLと、アピールしたい方向性を入力（最大3商品）
3. レコメンドバナーのリンクURL・画像URLを確認（前回保存した内容が自動で入っています）
4. 「商品情報を取得してタイトル・本文を作成」を押す
   → 商品ページを読み取り、Claudeがタイトルと本文の下書きを作成します
5. 下書きを確認・修正
6. 「JSONとテキストを生成する」を押して、それぞれダウンロード

バナーの内容を変えたときは「この内容を次回のデフォルトとして保存」を押しておくと、次回から自動で引き継がれます。

## 手元で試す（任意）

```bash
pip install -r requirements.txt
streamlit run app.py
```

ブラウザで `http://localhost:8501` が開きます。

## Anthropic APIキーについて

商品タイトル・本文の自動生成にはAnthropicのAPIキーが必要です。
- 手元で試す場合：サイドバーに直接入力できます
- streamlit.appにデプロイする場合：Streamlit Cloudの「Secrets」機能に以下の形式で登録してください

```toml
ANTHROPIC_API_KEY = "sk-ant-..."
```

APIキーが未設定の場合でもツールは動作しますが、その場合は商品ページから拾える情報（タイトル・説明文）をそのまま仮の内容として使うので、生成後に手動で編集してください。

## streamlit.app（Streamlit Community Cloud）へのデプロイ手順

1. **GitHubにリポジトリを作る**
   - GitHubアカウントで新しいリポジトリを作成（Public推奨、Privateでも可）。
   - このフォルダの中身（`app.py`、`requirements.txt`）をアップロードします。
     - GitHubの画面から「Add file → Upload files」でドラッグ＆ドロップでもOKです。
   - `banner_defaults.json` はアプリ起動後に自動生成されるので、最初はアップロード不要です。

2. **Streamlit Community Cloudにログイン**
   - https://share.streamlit.io/ にアクセスし、GitHubアカウントでログインします。

3. **Secretsを設定**
   - アプリ作成時（または作成後の「Settings → Secrets」）で、上記のAnthropic APIキーを登録します。

4. **アプリをデプロイ**
   - 「New app」を選択。
   - Repository：作成したリポジトリを選択。
   - Branch：`main`
   - Main file path：`app.py`
   - 「Deploy」をクリック。

5. 数分後、`https://（アプリ名）.streamlit.app` のようなURLでアプリが公開されます。

## 注意点

- レコメンドバナーの「引き継ぎ」は、アプリのファイルシステムに保存する方式です。
  同じデプロイが動き続けている間は引き継がれますが、アプリを再デプロイ（コードを更新して再度Deploy）すると
  リセットされる場合があります。長期的に確実に引き継ぎたい場合は、`banner_defaults.json` の内容を
  都度リポジトリにコミットして更新する運用にしてください。
- JSON構造は `crm_html_body_export.json`（MJML形式のCRMテンプレート）の
  「ヘッダー＋会員情報＋商品ブロック＋お礼メッセージ＋レコメンドバナー＋フッター」の構成に準拠しています。
  「お中元ランキング」のような一回限りの特殊セクションは含まれていないので、必要な場合は
  生成後のJSONに手動で追加してください。
- 商品ページによっては、Bot対策などでスクレイピング（`fetch_product_info`）に失敗することがあります。
  その場合はタイトル・画像URLが空になるので、下書き編集画面で手動入力してください。
